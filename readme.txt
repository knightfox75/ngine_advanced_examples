--------------------------------------------------------------------------------

	N'gine Advanced Examples
	Conjunto de ejemplos avanzados para N'gine.

--------------------------------------------------------------------------------

1. El objetivo
   El principal objetivo de estos ejemplos es mostrar el uso de funciones más
   avanzadas y posibilidades de uso del motor N'gine.
   
2. Que se incluye
   Ejemplos completos de:
   - Path finding (recursivo)
   - Aventura conversacional
   
3. A quién va destinado
   Estos ejemplos van destinados a los usuarios con conocimientos previos de
   programación en C/C++ o lenguajes similares y quieran adentrarse en el uso
   de este motor.
